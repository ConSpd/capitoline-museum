Display of Museum Exhibits

Once the user enters the areas where the various exhibits from the cultures of Greece, Egypt and Rome are displayed.

As it seems there is information:
  • The name of the exhibit.
  • A brief description.
  • The name of the exhibit.
  • A brief description of the exhibit.

    It is possible by pressing the "P" key, the user can add a copy to his/her cart and by pressing the "E" key, the user can see a more complete description of the exhibit. 

    By placing the user in front of the exhibit and pressing the "E" button, the user can see the more detailed description of the exhibit.

    The specific options and the functionality of each button can be viewed by pressing the "H" key, which brings the Help window to the foreground.
